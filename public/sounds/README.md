# 音声ファイルの配置

このディレクトリに以下の音声ファイルを配置してください：

## 必要な音声ファイル

1. **thinking-bgm.mp3** - シンキングタイム中のBGM（ループ再生）
   - 推奨: 3分程度のループ可能な音楽
   - 音量: 控えめに設定（後で調整可能）

## オプションの音声ファイル

以下のファイルを配置すると、より高品質な音声体験が可能です：

2. **thinking-start.mp3** - シンキングタイム開始の音
3. **thinking-end.mp3** - シンキングタイム終了の音

## 使用方法

1. 音声ファイルをこのディレクトリ（`public/sounds/`）に配置
2. ファイル名は上記の通りにしてください
3. ブラウザをリロードすると、音声ファイルが使用されます

## 注意事項

- 音声ファイルが存在しない場合は、Web Audio APIで生成された音が使用されます
- 音声ファイルの形式は MP3、OGG、WAV など、ブラウザがサポートする形式を使用してください
- ファイルサイズに注意してください（大きすぎると読み込みが遅くなります）
